## ✨ What's in v1.7.0

- **Spectral Unmixing** — auto-build reference matrix from single-color controls, per-fluorochrome peak detector override, linear + NNLS algorithms, SpectroFlo-compatible CSV save/load. Industry-standard pipeline for full-spectrum cytometers.
- **UMAP** — biex + z-score per-channel preprocessing, optional seed for bit-deterministic reproducible embeddings, auto density display mode.
- **Histogram** — Gaussian KDE smoothed curve, %max / count / log Y modes, median + mean stats overlay, drag-resize interval gates.
- **Auto-Fit Gates** ("Snap to Cluster") for ellipse, rectangle and polygon — one click.
- **FCS 2.0 / 3.0 / 3.1 / 3.2** file format support.
- **FlowJo workspace (.wsp)** import with corrupted-file recovery.
- **Workspace v2 (.fvw)** persists UMAP coordinates and spectral reference matrix across save/load.
- **49 % smaller bundle** via lazy-loaded chunks.

Full release notes: [v1.7.0 release page](https://github.com/Flowvision26/flowvision-releases/releases/tag/v1.7.0).
